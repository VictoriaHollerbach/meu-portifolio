    DICIONÁRIO DE ELEMENTOS HTML

    Abaixo estão as tags utilizadas no desenvolvimento deste projeto e suas funções:


    • <html>: Define a raiz de um documento HTML.
    • <head>: Contém metadados sobre o documento (título, links para CSS).
    • <body>: Define o corpo do documento, onde fica o conteúdo visível.
    • <header>: Define um cabeçalho para o documento ou uma seção.
    • <nav>: Define um conjunto de links de navegação.
    • <menu>: Utilizado para agrupar itens de menu.
    • <main>: Define o conteúdo principal e exclusivo do documento.
    • <section>: Define uma seção genérica em um documento.
    • <aside>: Define conteúdo lateral, relacionado ao conteúdo ao redor.
    • <footer>: Define o rodapé de uma página ou seção.
    • <h1>, <h2>, <h3>: Definem títulos de diferentes níveis de importância.
    • <p>: Define um parágrafo de texto.
    • <ul>: Define uma lista não ordenada (com marcadores).
    • <li>: Define um item dentro de uma lista.
    • <a>: Cria hiperlinks para outras páginas ou seções.
    • <img>: Utilizada para inserir imagens no documento.
    • <div>: Define uma divisão ou bloco genérico para estilização.
    • <b>: Aplica negrito ao texto para destaque visual.
